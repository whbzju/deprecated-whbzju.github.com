收集我个人学习使用Vim的小技巧

##快速替换##

* 语法: [addr]s/源字符串/目的字符串/[option]
* [addr] 表示检索范围，省略时表示当前行。如：
	* “1，20” ：表示从第1行到20行；
	* “%” ：表示整个文件，同“1,$”；
	* “. ,$” ：从当前行到文件尾；
* s：正则表达式中替换命令
* [option] : 表示操作类型
	* g 表示全局替换; 
	* c 表示进行确认
	* p 表示替代结果逐行显示（Ctrl + L恢复屏幕）
	* i 不区分大小写
	* 省略option时仅对每行第一个匹配串进行替换；
* 如果在源字符串和目的字符串中出现特殊字符，需要用”\”转义

例子：将多个空格换成一个空格: %s/  */ /g

##多文件打开
打开：split file  
切换窗口：双击ctrl+w，或ctrl+w+上下左右键
